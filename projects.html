Plane Management System


February 2024 - March 2024

In this project, I developed a ticket booking system using Java, which includes multiple functionalities,

  1. Buy a seat
  2. Cancel a seat
  3. Find first available seat
  4. Show seating plan
  5. Print ticket information and total sales
  6. Search ticket

The project also incorporates object-oriented concepts by introducing two classes: Person and Ticket. The Person class stores user information such as name, surname, and email, while the Ticket class tracks seat details, price, and is associated with a Person object. An array of Ticket objects stores all sold tickets for easy access and management.

Additional features include displaying the seating plan, searching for ticket information, printing sold ticket details with total price calculations, and saving ticket information to text files. This project demonstrates my ability to apply control structures, work with classes and objects, and manage file I/O operations in Java.

Student Progression Predictor


November 2024 - December 2024

This Python-based program predicts academic progression outcomes for students based on credit inputs in pass,defer, and fail categories.

It efficiently handles multiple inputs with validation to ensure accurate credit totals and appropriate progression categories, such as Progress, Trailing, Module Retriever,and Exclude.

The program also generates a visual histogram using the graphics.py module, allowing users to see the distribution of outcomes. Additionally, it stores data in lists and text files for further analysis, making it a robust and versatile tool for academic staff.
